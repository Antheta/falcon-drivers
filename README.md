# Drivers for Falcon

Read more about Falcon drivers [here](https://docs.antheta.com/falcon/drivers).

## Prerequisites

[Falcon](https://github.com/Antheta/falcon-php) is required for using any of these drivers.


## Installation
```bash
composer require antheta/falcon-drivers
```
